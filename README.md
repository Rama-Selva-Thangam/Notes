# Notes App

The **Notes App** is a simple web application that allows users to create, edit, and manage notes. It provides a clean and intuitive user interface for taking and organizing notes.

## Features

- **Create and Edit Notes:** Users can create new notes by providing a title and content. Existing notes can be edited, and changes are saved automatically.

- **Rich Text Formatting:** The app supports basic text formatting options such as bold, italic, underline, and strikethrough, providing users with flexible note-taking capabilities.

- **Search Functionality:** A search bar allows users to search for notes based on their titles. The search feature helps users quickly locate specific notes within their collection.

- **Save and Delete Notes:** Users can save their notes, and the app automatically manages the last edited timestamp. Additionally, notes can be deleted when no longer needed.

- **Word Count and File Information:** The app displays information about the current note, including the file name, word count, and last edited timestamp.

## Technologies Used

- **HTML, CSS, JavaScript:** The app is built using standard web technologies for the frontend, ensuring compatibility across different browsers.

- **Font Awesome:** Icons from the Font Awesome library are used to enhance the visual representation of buttons and elements within the app.

## How to Use

1. Open the `index.html` file in a web browser to launch the application.

2. Use the left panel to view and search existing notes. Clicking on a note in the list populates the title and content in the middle panel for editing.

3. In the middle panel, provide a title and content for a new note or edit an existing note.

4. Use the formatting buttons (bold, italic, underline, strikethrough) to apply text formatting as needed.

5. Click the "Save Note" button to save changes or create a new note.

6. Click the "Delete Note" button to remove the currently displayed note.

7. The right panel displays additional information about the note, such as the file name, word count, and last edited timestamp.

## Project Structure

- **index.html:** The main HTML file that defines the structure of the web page.

- **style.css:** The stylesheet containing the visual styling for the application.

- **script.js:** The JavaScript file that handles the dynamic functionality and interactivity of the app.
